Text translation via ID

In this paragraph only the phrase Hello World will be translated upon change of language. This is done by assigning it an id shared between all of it's translations.

Text translation via Content

In this paragraph the words dog and cat will be translated upon change of language. Again like the previous example, this is done by shared ids across all translations, with the exception that the word itself is used as the id.

Text translation from external source

This entire paragraph will be translated uponchange of language. This is done by fetching the translated data from an external source (in this case a GET request to the server) when it is needed.
